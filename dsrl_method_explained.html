Steering a Diffusion Policy with RL — dsrl_demo.py explained

A function-by-function companion to the hopper demo notebook · for the lecture

DSRL (Diffusion Steering via Reinforcement Learning, Wagenmaker et al., CoRL 2025) fine-tunes a diffusion/flow policy without changing its weights. A diffusion policy turns an initial noise vector into an action by iterative denoising. DSRL freezes that denoiser and trains a small RL agent (SAC-style) whose action is the initial noise. Steering the noise steers the behavior.

This file, dsrl_demo.py, is the thin layer the demo notebook calls into. Below, each function is shown with its real code and explained in terms of the method.

def load_config(overrides=None):
    """Load the project's hopper config without Hydra and apply overrides."""
    OmegaConf.register_new_resolver("eval", eval, replace=True)
    OmegaConf.register_new_resolver("round_up", math.ceil, replace=True)
    OmegaConf.register_new_resolver("round_down", math.floor, replace=True)
    OmegaConf.register_new_resolver("now", lambda *a: "demo", replace=True)

    cfg = OmegaConf.load(os.path.join(BASE, "cfg/gym/dsrl_hopper.yaml"))
    cfg.use_wandb = False
    if overrides:
        cfg.merge_with(OmegaConf.create(overrides))
    OmegaConf.resolve(cfg)

    # Make the repo-relative ("./dppo/...") paths absolute so the demo works
    # regardless of the current working directory.
    def _abs(p):
        return p if os.path.isabs(p) else os.path.normpath(os.path.join(BASE, p))
    cfg.normalization_path = _abs(cfg.normalization_path)
    cfg.base_policy_path = _abs(cfg.base_policy_path)
    cfg.model.network_path = _abs(cfg.model.network_path)
    cfg.dppo_path = _abs(cfg.dppo_path)
    return cfg

2 · Environments

The diffusion policy emits an action chunk (several timesteps at once), and it expects observations normalized the same way as during pretraining. These helpers build exactly that env stack.

_make_chunk_env

def _make_chunk_env(cfg):
    """A single training env: normalized obs + action-chunk wrapper (matches train_dsrl)."""
    env = gym.make(cfg.env_name)
    env = ObservationWrapperGym(env, cfg.normalization_path)
    env = ActionChunkWrapper(env, cfg, max_episode_steps=cfg.env.max_episode_steps)
    return env
What. gym.make(hopper) → ObservationWrapperGym (normalizes obs to [-1,1] and un-normalizes actions using the pretraining stats) → ActionChunkWrapper (plays the act_steps actions of a chunk sequentially). Method link. Matching the pretraining normalization is essential — the frozen denoiser only behaves well on inputs it recognizes.

3 · The model — this is the core of DSRL

make_model builds the SAC-style agent in the noise space and hands it the frozen diffusion policy. This is where “steer the noise” becomes code.

make_model

def make_model(cfg, env):
    """Build the DSRL (DSRL-NA) algorithm exactly as train_dsrl.py does."""
    post_linear_modules = [torch.nn.LayerNorm] if cfg.train.use_layer_norm else None
    net_arch = [cfg.train.layer_size] * cfg.train.num_layers
    policy_kwargs = dict(
        net_arch=dict(pi=net_arch, qf=net_arch),
        activation_fn=torch.nn.Tanh,
        log_std_init=0.0,
        post_linear_modules=post_linear_modules,
        n_critics=cfg.train.n_critics,
    )
    base_policy = load_base_policy(cfg)
    model = DSRL(
        "MlpPolicy", env,
        learning_rate=cfg.train.actor_lr,
        buffer_size=10_000_000,
        learning_starts=1,
        batch_size=cfg.train.batch_size,
        tau=cfg.train.tau,
        gamma=cfg.train.discount,
        train_freq=cfg.train.train_freq,
        gradient_steps=cfg.train.utd,
        ent_coef="auto" if cfg.train.ent_coef == -1 else cfg.train.ent_coef,
        target_update_interval=1,
        target_entropy="auto" if cfg.train.target_ent == -1 else cfg.train.target_ent,
        verbose=0,
        policy_kwargs=policy_kwargs,
        diffusion_policy=base_policy,
        diffusion_act_dim=(cfg.act_steps, cfg.action_dim),
        noise_critic_grad_steps=cfg.train.noise_critic_grad_steps,
        critic_backup_combine_type=cfg.train.critic_backup_combine_type,
    )
    return model, base_policy
What. Builds a DSRL agent (a SAC variant) with a 3-layer MLP actor/critic. load_base_policy(cfg) returns the frozen diffusion policy (DPPO DiffusionEval, DDIM sampling, controllable initial noise). Method link — read these arguments:
  • diffusion_policy=base_policy — the frozen denoiser the agent steers (never trained).
  • diffusion_act_dim=(act_steps, action_dim) — the shape of the noise vector = the shape of the action chunk. The actor's action is that noise.
  • noise_critic_grad_steps & critic_backup_combine_type — the “NA” variant (DSRL-NA): an auxiliary critic in the noise space, distilled from a Q-function learned in the original action space (more sample-efficient).
So the actor proposes z; before z reaches the environment it is denoised by the frozen policy into an action chunk.

def warmup_buffer(model, cfg, steps=200):
    """Seed the replay buffer with rollouts of the (un-steered) base diffusion policy."""
    if steps <= 0:
        return
    env = make_train_env(cfg)
    collect_rollouts(model, env, steps, model.diffusion_policy, cfg)
    env.close()
What. Pre-fills the replay buffer with rollouts of the un-steered base policy (collect_rollouts). Method link. A behavior prior — RL starts from sensible coverage instead of pure randomness, speeding up learning.

4 · Rollouts & rendering — “before vs after”

The same episode loop drives both clips; only the action function differs. That single difference is the before/after story.

_render_rollout

def _render_rollout(cfg, action_fn, max_chunks=250, seed=0):
    """Step a single rendered hopper env, choosing each action chunk via action_fn(obs)."""
    base = gym.make(cfg.env_name)
    base.seed(seed)
    env = ObservationWrapperGym(base, cfg.normalization_path)
    obs = env.reset()
    frames, total = [], 0.0
    for _ in range(max_chunks):
        chunk = action_fn(obs).reshape(cfg.act_steps, cfg.action_dim)
        done = False
        for i in range(cfg.act_steps):
            obs, r, done, _ = env.step(chunk[i])
            total += float(r)
            frames.append(base.render(mode="rgb_array"))
            if done:
                break
        if done:
            break
    base.close()
    return frames, total
What. Runs one episode on a rendered env. Each step it calls action_fn(obs) to get an action chunk, plays it, and stores RGB frames.

rollout_base_policy

def rollout_base_policy(base_policy, cfg, max_chunks=250, seed=0):
    """Behavior of the pretrained diffusion policy with standard Gaussian noise (the 'before')."""
    dev = cfg.model.device

    def action_fn(obs):
        noise = torch.randn(1, cfg.act_steps, cfg.action_dim, device=dev)
        a = base_policy(torch.tensor(obs[None], device=dev, dtype=torch.float32), noise)
        return np.asarray(a)

    return _render_rollout(cfg, action_fn, max_chunks, seed)
BEFORE. action_fn samples z ~ N(0, I) and denoises it: base_policy(obs, noise). This is a vanilla diffusion policy with noise from the prior — no steering.

rollout_dsrl

def rollout_dsrl(model, cfg, max_chunks=250, seed=0, deterministic=True):
    """Behavior of the DSRL-fine-tuned (steered-noise) policy (the 'after')."""
    def action_fn(obs):
        a, _ = model.predict_diffused(obs[None].astype(np.float32),
                                      deterministic=deterministic)
        return np.asarray(a)

    return _render_rollout(cfg, action_fn, max_chunks, seed)
AFTER. action_fn calls model.predict_diffused(obs): the learned actor picks z, then the same frozen denoiser turns it into an action. Same denoiser, smarter noise → better behavior.

5 · Measuring the improvement

A single rollout is noisy. These run many episodes (no rendering) and summarize with confidence intervals — the rigorous “is it really better?”.

eval_returns_base

def eval_returns_base(base_policy, cfg, n_episodes=50, max_chunks=300, seed=1000):
    """Returns of the pretrained diffusion policy with standard Gaussian noise."""
    dev = cfg.model.device

    def action_fn(obs):
        noise = torch.randn(obs.shape[0], cfg.act_steps, cfg.action_dim, device=dev)
        a = base_policy(torch.tensor(obs, device=dev, dtype=torch.float32), noise)
        return np.asarray(a).reshape(obs.shape[0], cfg.act_steps * cfg.action_dim)

    return _eval_returns(cfg, action_fn, n_episodes, max_chunks, seed)
What. Same idea as the rollouts but headless and over many episodes. eval_returns_base uses N(0,I) noise; eval_returns_dsrl uses the learned actor (deterministic or stochastic).

summarize

def summarize(name, rets, lens):
    m, sd = float(rets.mean()), float(rets.std(ddof=1)) if len(rets) > 1 else 0.0
    sem = sd / np.sqrt(len(rets)) if len(rets) else 0.0
    return dict(name=name, n=len(rets), mean=m, std=sd, sem=sem,
                ci95=1.96 * sem, median=float(np.median(rets)),
                min=float(rets.min()), max=float(rets.max()), mean_len=float(lens.mean()))
What. Reduces a list of episode returns to mean, std, 95% CI, median and mean length. Method link. Non-overlapping CIs between base and DSRL ⇒ a statistically clear gain (our run: ~+56%).

6 · Training-metrics callback

Plugs into model.learn(...) to record what the curves in the notebook show.

DemoCallback

class DemoCallback(BaseCallback):
    """Collects training losses and periodic deterministic-eval returns for plotting."""

    def __init__(self, cfg, eval_every=2000, n_eval_episodes=5, max_chunks=250):
        super().__init__()
        self.cfg = cfg
        self.eval_every = eval_every
        self.n_eval_episodes = n_eval_episodes
        self.max_chunks = max_chunks
        self.act_chunk = cfg.act_steps
        self.n_envs = cfg.env.n_envs
        # logs
        self.t_loss, self.actor_loss, self.critic_loss, self.noise_critic_loss = [], [], [], []
        self.ent_coef = []
        self.t_eval, self.eval_return = [], []
        self.train_t, self.train_return = [], []
        self._eval_env = None
        self._next_eval = 0
        self._env_steps = 0

    def _env_timesteps(self):
        # number of underlying (per-action) env steps seen so far
        return self._env_steps

    def _on_step(self):
        self._env_steps += self.act_chunk * self.n_envs
        for info in self.locals["infos"]:
            if "episode" in info:
                self.train_t.append(self._env_steps)
                self.train_return.append(float(info["episode"]["r"]))
        # record losses from the SB3 logger
        v = self.model.logger.name_to_value
        if "train/actor_loss" in v:
            self.t_loss.append(self._env_steps)
            self.actor_loss.append(v.get("train/actor_loss", np.nan))
            self.critic_loss.append(v.get("train/critic_loss", np.nan))
            self.noise_critic_loss.append(v.get("train/noise_critic_loss", np.nan))
            self.ent_coef.append(v.get("train/ent_coef", np.nan))
        # periodic deterministic eval
        if self._env_steps >= self._next_eval:
            self._next_eval = self._env_steps + self.eval_every
            self.eval_return.append(self._evaluate())
            self.t_eval.append(self._env_steps)
            print(f"[eval] env_steps={self._env_steps:>7d}  mean_return={self.eval_return[-1]:8.1f}")
        return True

    def _evaluate(self):
        if self._eval_env is None:
            self._eval_env = make_eval_env(self.cfg, n_envs=1)
        env = self._eval_env
        returns = []
        for _ in range(self.n_eval_episodes):
            obs = env.reset()
            ep_ret, done = 0.0, False
            for _ in range(self.max_chunks):
                action, _ = self.model.predict_diffused(obs, deterministic=True)
                obs, reward, done, _ = env.step(action)
                ep_ret += float(reward[0])
                if done[0]:
                    break
            returns.append(ep_ret)
        return float(np.mean(returns))
What. On every SB3 step it logs the losses (actor_loss, critic_loss, the DSRL-specific noise_critic_loss, ent_coef) and, every eval_every env-steps, runs a short deterministic evaluation and stores the mean return. Method link. The eval-return curve is the headline metric (task reward rising); the losses show learning dynamics, not performance.

Key takeaways for the audience

  • Frozen denoiser, learned noise. DSRL never updates the diffusion policy — it learns which initial noise to feed it.
  • The actor lives in noise space. Its action has the same shape as the action chunk and is denoised before reaching the env (predict_diffused).
  • Before = N(0,I); After = learned z. Only action_fn changes between the two clips.
  • DSRL-NA adds a noise-space critic distilled from an action-space Q-function for sample efficiency.
  • Read it as reward. Judge success by the eval-return curve and the multi-episode bar chart (mean ± 95% CI), not by the loss values.
